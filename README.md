# NLP Learn

NLP concepts implemented from scratch for learning purposes.

## Contents

| File | Topic |
|------|-------|
| `main.py` | Bag-of-Words vectors, cosine similarity |
| `NaiveBayes_LogisticRegression.py` | Text classification on 20 Newsgroups (CountVectorizer + NB + LR) |
| `Embedings.py` | Count-based word embeddings: co-occurrence matrix → PPMI → SVD → nearest neighbors |

## Requirements

```bash
pip install numpy scikit-learn
```
